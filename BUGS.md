# User unfriendly focus behavior when using the panels

Text input loses focus when opening or closing the panels, this is annoying in mobile since this closes the mobile keyboard, when focusing the input the foucs should be kept when navigating the panels

# Drag and drop behavior

When moving an item up the list it triggers when touching the upper element, when moving down it triggers very late, we have to find a middle ground so both directions behave similar

When dropping the element it renders back to the original place and then to the target place a few ms later
